# Reading Profile — The Mom Test

**体裁：** 创业访谈方法论（短章、对话驱动）  
**风格标杆：** `books/TheMomTest/02.Chapter-1.md`

## 阅读单位

- 按 `sections.json` 一章一节；单章约 10–20 页，**不必再拆**。
- Conclusion 含 cheatsheet，笔记中单独提炼 **可复用清单**。

## 笔记侧重

| 优先 | 说明 |
|------|------|
| **规则与反例** | 妈妈测试三规则、坏问题 vs 好问题 |
| **对话对照** | 同一情境两种问法 → 表格（差问法 / 好问法 / 学到什么） |
| **可执行标准** | 「有用对话」的判断标准、承诺与推进信号 |
| **误区** | 假阳性、过早推销、礼貌性恭维 |

## 笔记结构（在通用模板之上）

```markdown
## 核心总结
### <规则或主题>
> 用中文说清楚：…
## 种子 / 伏笔
## 案例卡片
## 与已读部分的呼应
## 阅读提示
```

- **核心总结**：规则放前面；复杂对话用 **表格** 不用长段落复述
- **案例卡片**：分 **失败对话** / **通过对话**；标 **人物、问法、后果、原则**
- **种子**：记 **规则、清单项、待组合的概念**（非剧情线）；如「三规则」「坏数据类型」
- **呼应**：与前几章规则叠加关系（Ch3 问法建立在 Ch1–2 之上）

## 术语

- 书名规则保留英文：**妈妈测试（The Mom Test）**
- 方法术语首次对照：假阳性（false positive）、承诺（commitment）、推进（advancement）等

## 阅读提示（每章末尾）

1. 本章新增/强化了哪条规则  
2. 一张「下次访谈前看一眼」的小表或三 bullet  
3. 与 cheatsheet（Conclusion）的对应关系（若已读）  
4. 下一章预告  

## 不适用

- 不必写全书级哲学论证链  
- 不必神话/跨文化对照  
- 人物线弱，案例卡片重 **对话范式** 而非人物命运
